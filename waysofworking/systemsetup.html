System Setup

You have to run Linux or OSX when working at Western Cape Labs. If you run Linux, we highly recommend that you use a Debian-based distribution like Ubuntu or Linux Mint - you're on your own if you use something else.

Useful Software

Below are really useful things to install on a new system with some basic Linux installation instructions. As installation procedures change over time you might have to do something a bit differently.

Git

We use Git for software version control of all our projects. Github stores our code remotely. If you don't have a Github account, go make one now! If you have one already, make sure your email addresses are setup correctly - including adding your @westerncapelabs.com email address. Then do:

sudo apt-get install git
git config --global user.name _yourgithubusername_
git config --global user.email _youremail_

You can read up on setting up SSH Authentication, but to get started quickly you can just use HTTPS Authentication. However, to prevent having to enter your username and password every time, use a credential helper to set how often you get password-prompted to every 24 hours (86400 seconds)

git config --global credential.helper cache
git config --global credential.helper 'cache --timeout=86400'

Text Editor

Our two favourite editors are Sublime Text 3 and Atom. Sublime Text is loved by just about everyone who has used it, but it kind of costs $70, while Atom is free and is practically a clone of Sublime Text. Sublime Text kind of costs $70 because you can try it for free forever, you just have to ignore a pop-up 'buy me!' message every now and then. Or convince Mike to buy it for you :)
